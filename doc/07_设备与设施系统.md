# 设备与设施系统

## 一、设施系统总览

### 1.1 设施列表与功能

俱乐部设施是基地经营的核心构成，每个设施提供独特加成。

| 设施 | 基础功能 | 解锁条件 | 最大等级 |
|------|----------|----------|----------|
| **作战指挥室** | 决定基地等级上限，控制其他设施解锁 | 初始解锁 | Lv50 |
| **直播室** | 自动产生游戏币收入（放置收益） | 初始解锁 | Lv50 |
| **训练室** | 提升员工技术培养速度 | 基地Lv2 | Lv50 |
| **装备库** | 装备存储、强化成功率加成 | 基地Lv3 | Lv50 |
| **VIP接待室** | 提升委托人满意度、解锁高级委托 | 基地Lv4 | Lv30 |
| **车库** | 解锁车型、提升撤离成功率 | 基地Lv5 | Lv30 |
| **情报中心** | 生成情报碎片、提升搜刮效率 | 基地Lv6 | Lv30 |
| **休息室** | 恢复员工心情、好感度自然增长 | 基地Lv3 | Lv30 |
| **仓库** | 道具/装备容量上限 | 初始解锁 | Lv50 |
| **荣誉墙** | 展示成就、提供全局属性加成 | 基地Lv10 | Lv20 |
| **医疗室** | 员工委托受伤后加速恢复 | 基地Lv7 | Lv20 |
| **战术研究室** | 解锁团队战术Buff、降低培养消耗 | 基地Lv12 | Lv20 |

### 1.2 设施升级框架

所有设施遵循统一升级公式：

```
升级消耗（俱乐部资金）= Base_Cost × (1.25)^Level
升级时间 = Base_Time × (1.3)^Level （上限48小时）
前置条件 = 作战指挥室等级 + 特定基地等级
```

## 九、设施维护/耐久度系统

### 9.1 耐久度机制

每个设施拥有100点耐久度，随时间和使用消耗：

| 设施 | 耐久度消耗 | 消耗速度 |
|------|-----------|----------|
| 直播室 | 持续运作消耗 | 5点/天 |
| 训练室 | 训练中消耗 | 1点/次训练 |
| 装备库 | 强化时消耗 | 0.5点/次强化 |
| 车库 | 委托使用消耗 | 2点/次委托 |
| 其他设施 | 自然老化 | 2点/天 |

### 9.2 耐久度影响

| 耐久度区间 | 效果衰减 |
|-----------|----------|
| 100-71% | 正常运作 |
| 70-41% | 效果-15% |
| 40-11% | 效果-30% |
| 10-1% | 效果-50% |
| 0% | 停止运作（需维修） |

### 9.3 维修消耗

```
维修消耗（俱乐部资金）= (100 - 当前耐久度) × 设施等级 × 200

一键全部维修：总消耗 × 0.9（9折优惠）
自动维修：VIP特权，耐久度低于30%自动维修
```
