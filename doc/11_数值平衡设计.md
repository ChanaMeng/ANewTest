# 数值平衡设计

## 一、数值设计总纲

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **对数曲线** | 核心养成采用对数增长，前期快后期慢，制造持续追求 |
| **多来源解耦** | 战斗力来自多个系统，避免单一系统绑架 |
| **软上限设计** | 不设硬天花板，但边际收益递减 |
| **经济可控** | 每日资源产出精确到±10%误差 |
| **商业不绑架** | 付费加速≤2x，不付费也能在1.5倍时间内达到相同进度 |

### 1.2 核心公式总览

```
战斗力 = [（枪法×2 + 反应×1.5 + 生存×1.5 + 感知×1 + 指挥×1 + 魅力×0.5） 
         ÷ 100] 
         × 技术系数 
         × 稀有度倍率 
         × (1 + 设施加成) 
         × (1 + 好感度加成) 
         × (1 + 联盟科技加成)

委托成功率 = 基础成功率 + Σ（员工属性-委托需求属性）÷ 100 + 设施加成 + 车辆加成 - 委托难度系数

直播收益/小时 = 直播室产出基数 × (1 + Σ入驻员工技术/200) × (1 + Σ入驻员工魅力/300) × 人气系数

人气 = Σ（自然增长 + 活动加成 + 设施加成）× (1 - 衰减率)
```

## 三、护送委托难度与收益公式

### 3.1 委托难度等级

| 难度 | 等级星星 | 推荐技术要求 | 基础成功率 | 基础奖励 |
|------|----------|-------------|-----------|----------|
| 新手 | ⭐ | 技术Lv1 | 95% | 500-1,000 信用点 |
| 普通 | ⭐⭐ | 技术Lv10 | 85% | 1,000-3,000 信用点 |
| 进阶 | ⭐⭐⭐ | 技术Lv25 | 70% | 3,000-8,000 信用点 |
| 困难 | ⭐⭐⭐⭐ | 技术Lv45 | 55% | 8,000-20,000 信用点 |
| 精英 | ⭐⭐⭐⭐⭐ | 技术Lv70 | 40% | 20,000-50,000 信用点 |
| 史诗 | ⭐⭐⭐⭐⭐⭐ | 技术Lv100 | 25% | 50,000-120,000 信用点 |
| 传说 | 🌟 | 技术Lv150 | 10% | 120,000-300,000 信用点 |

### 3.2 委托成功率公式

```
成功率 = Base Rate 
       + (队伍平均技术等级 - 委托要求技术等级) × 0.5%
       + (队伍平均属性 - 委托要求属性) × 0.3%
       + 设施加成（车库/作战指挥室等）
       + 车辆加成
       - 委托难度修正

公式约束：
  - 成功率上限 = 98%（永远有翻车风险）
  - 成功率下限 = 5%（永远有机会）
  - 超出属性部分的加成减半

属性匹配加成：
  每个满足的属性要求（高于委托需求的属性）= +2%成功率
  配置符合委托"推荐定位"的员工 = +5%/人
```

### 3.3 委托收益公式

```
委托收益 = 基础奖励 × 成功率修正 × 满意度修正

成功率修正：
  成功时 = 基础奖励 × (1 + 溢出属性 × 0.1%)
  失败时 = 基础奖励 × 30%（安慰奖）

满意度修正（取决于气氛组员工魅力和VIP接待室）：
  满意度 = 50% + 队伍魅力值/200 + VIP接待室加成
  收益修正 = 0.7 + 满意度 × 0.6
  （范围：0.85 ～ 1.27）

实际收益示例：
  - 低配置成功：基础奖励 × 0.85
  - 刚好配置成功：基础奖励 × 1.0
  - 豪华配置成功：基础奖励 × 1.20
  - 任何配置失败：基础奖励 × 0.30
```

## 四、直播收益公式

### 4.1 基础产出公式

```
时产游戏币 = 直播室Lv产出基数 
           × (1 + Σ入驻员工技术 × 0.005)
           × (1 + Σ入驻员工魅力 × 0.003)
           × 人气系数
           × 双倍活动系数（如有）

直播室Lv产出基数表（游戏币/小时）：
  Lv1:  100
  Lv5:  500
  Lv10: 1,500
  Lv20: 5,000
  Lv30: 15,000
  Lv40: 35,000
  Lv50: 80,000
```

### 4.2 多员工入驻公式

```
当直播室入驻N名员工时：

总产出 = 直播室产出基数 × (1 + Σ(员工技术 × 0.005)) 
                                × (1 + Σ(员工魅力 × 0.003))

入驻员工产出权重：
  - 气氛组 🎤：基础产出 × 1.3（魅力加成）
  - 全能王 ⭐：基础产出 × 1.1（技术+魅力双加成）
  - 枪男 🔫：基础产出 × 1.0（仅技术加成）
  - 老六 🥷：基础产出 × 0.9（领域不符）
  - 车头 🚗：基础产出 × 0.85（领域不符）

示例计算（直播室Lv20，入驻3人）：
  员工A「吴姐」★★★★★ 气氛组 技术Lv65 魅力120
  员工B「锁子哥」★★★★★ 枪男 技术Lv85 魅力60
  员工C「塔科夫老兵」★★★★★ 全能王 技术Lv72 魅力90

  技术加成 = 1 + (65+85+72) × 0.005 = 1 + 1.11 = 2.11
  魅力加成 = 1 + (120+60+90) × 0.003 = 1 + 0.81 = 1.81
  总产出 = 5,000 × 2.11 × 1.81 = 19,096 游戏币/小时
  加上配置系数 = 19,096 × (1.3+1.0+1.1)/3 = 19,096 × 1.13 = 21,578 游戏币/小时
```

## 五、人气系统公式

### 5.1 人气自然增长

```
基础人气增长 = 直播室等级 × 5 + 俱乐部等级 × 10 + Σ(员工魅力) × 2/天

示例：
  基地Lv10 + 直播室Lv15 + 5名平均魅力80的员工
  日增长 = 15×5 + 10×10 + 5×80×2 = 75+100+800 = 975人气/天
```

### 5.2 人气衰减公式

```
人气衰减率 = 5% / 天（基础）
           - 0.1% × VIP接待室等级（最高-3%）
           - 1% × (在线活跃天数/7)（每周减1%，满活跃-5%）

最终人气衰减率 = 5% - 3% - 5% = -3%（实际为2%下限）

人气平衡点（当日增长=当日衰减时的人气值）：
  人气平衡点 = 日增长 ÷ 衰减率
  例：日增975，衰减2%，平衡点 = 975 ÷ 0.02 = 48,750

人气系数（影响收益）：
  人气 < 1,000：系数 1.0
  1,000 ≤ 人气 < 5,000：系数 1.0 + (人气-1,000)/40,000 = 1.0~1.1
  5,000 ≤ 人气 < 10,000：系数 1.1 + (人气-5,000)/25,000 = 1.1~1.3
  10,000 ≤ 人气 < 50,000：系数 1.3 + (人气-10,000)/200,000 = 1.3~1.5
  50,000 ≤ 人气 < 100,000：系数 1.5 + (人气-50,000)/500,000 = 1.5~1.6
  100,000+：系数 1.6 + (人气-100,000)/1,000,000 = 1.6~1.7
```

### 8.3 战斗力公式（完整）

```
战斗力 = FLOOR(
  (枪法×2.0 + 反应×1.5 + 生存×1.5 + 感知×1.0 + 指挥×1.0 + 魅力×0.5)
  × 稀有度倍率
  × 技术系数
  × (1 + 装备加成%)
  × (1 + 专属装备加成%)
  × (1 + 好感度属性加成%)
  × (1 + 设施加成%)
  × (1 + 联盟科技加成%)
)

示例计算：★★★★★ Lv50枪男「锁子哥」
  六维：(120, 40, 70, 55, 38, 52)
  基础战斗值 = 120×2+70×1.5+55×1.5+40×1+38×1+52×0.5 
             = 240+105+82.5+40+38+26 = 531.5
  稀有度倍率 = 4.2
  技术Lv50系数 = 7.6
  装备加成 = 25%
  设施加成 = 15%
  战斗力 = 531.5 × 4.2 × 7.6 × 1.25 × 1.15 
          = 531.5 × 4.2 × 7.6 × 1.4375
          ≈ 24,400
```
