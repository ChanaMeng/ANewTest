# 《搜打撤俱乐部》游戏设计文档 V2

> **游戏类型**: 模拟经营 / 放置 / 养成 / 文字模拟  
> **目标平台**: 微信小游戏（竖屏）  
> **对标产品**: 《人生模拟器》《叫我大掌柜》《动物餐厅》

---

## 一、项目概述

### 1.1 游戏简介

《搜打撤俱乐部》是一款以"搜打撤"（Search-Fight-Extract）游戏生态为背景的**模拟经营放置手游**。

玩家扮演搜打撤俱乐部的经营者，招募各具特色的员工，通过**直播收益**（放置挂机）持续获取游戏币，再用游戏币**派遣护送任务**赚取俱乐部资金，最终将资金投入**设备升级、人气提升和员工培养**，打造圈内最具影响力的俱乐部。

### 1.2 V2 核心设计理念

| 设计理念 | 说明 |
|----------|------|
| **合法化定位** | 俱乐部 = 游戏服务平台，提供战术协助和娱乐内容，非灰色产业 |
| **双货币循环** | 游戏币（挂机产出）→ 护送任务消耗 → 俱乐部资金（运营收入）→ 俱乐部发展 |
| **放置为主、操作为辅** | 直播收益全自动挂机；护送任务手动派遣；经营决策手动干预 |
| **文字叙事驱动** | 事件日志系统承载随机事件，类似《人生模拟器》的沉浸体验 |
| **成长可视化** | 俱乐部规模、人气值、员工实力均有明确数值反馈 |

### 1.3 V2 与 V1 核心变更

| 维度 | V1 | V2 |
|------|-----|-----|
| **世界观** | 灰色地带打手基地 | 合法俱乐部服务平台 |
| **核心玩法** | 卡牌收集 + 委托派遣 | 放置挂机 + 模拟经营 + 养成 |
| **货币体系** | 单一货币 | 双货币：游戏币 + 俱乐部资金 |
| **收入来源** | 执行委托 | 直播收益（放置）+ 护送任务（派遣） |
| **养成维度** | 单一属性 | 技术（核心）+ 人气（辅助）+ 非技术属性 |
| **叙事方式** | 事件弹窗 | 事件日志（类似人生模拟器） |
| **平台适配** | 横屏/竖屏未定 | 微信小游戏竖屏 |
| **关键对标** | 电竞经理、大掌柜 | 动物餐厅、人生模拟器 |

### 1.4 核心体验

| 维度 | 描述 |
|------|------|
| **放置爽感** | 加入俱乐部后，直播收益自动产出游戏币；离线也有收益，每天上线收菜 |
| **经营决策** | 如何分配俱乐部资金——升级设备？提升人气？培养员工？每项选择都影响俱乐部发展路径 |
| **养成深度** | 员工技术培养采用递增成本机制，从菜鸟到王牌需要长期规划和投入 |
| **叙事沉浸** | 每日事件日志娓娓道来俱乐部日常，成功时的喜悦、翻车时的无奈，代入感极强 |
| **目标驱动** | 从初级护送任务起步，逐步解锁中级、高级护送任务，见证俱乐部成长壮大 |

### 1.5 世界观设定

在平行世界的游戏圈中，"搜打撤"是最热门的射击游戏品类。无数玩家在虚拟战场上搜刮、战斗、撤离。在这个过程中，一部分技术过硬的玩家自发组建俱乐部，为其他玩家提供战术指导和护送服务，同时通过直播分享搜打撤的精彩时刻。

玩家扮演的就是这样一家俱乐部的经营者。你的目标很简单——把这家小俱乐部经营成圈内最知名的服务平台。

### 1.6 核心循环

```
  直播收益（放置挂机）
       │
       ↓  持续产出
     游戏币 ──────────────→ 培养员工·技术
       │                         │
       │  消耗                    │ 技术越高
       ↓                         │ 护送成功率越高
  派遣护送任务                    │ 收益效率越高
   (初级/中级/高级)               │
       │                         │
       ↓  奖励                    │
    俱乐部资金 ←──────────────────┘
       │
       ├──→ 升级设备 → 提升人气 → 直播效率↑
       ├──→ 招募员工 → 扩大规模
       └──→ 培养非技术属性（魅力/耐力等）
```

### 1.7 核心循环文字说明

1. **直播收益（放置挂机）**：员工自动进行游戏直播，持续产出**游戏币**。人气越高，直播效率越高。离线也能产出，上线收取。
2. **派遣护送任务**：消耗**游戏币**派遣员工执行护送任务。分初级/中级/高级三档，高难度消耗更多游戏币、产出更多**俱乐部资金**，同时对员工**技术**有门槛要求。
3. **获取俱乐部资金**：护送任务奖励**俱乐部资金**。俱乐部资金是俱乐部发展的核心资源。
4. **升级与培养**：俱乐部资金用于升级设备（提升人气）、提升员工非技术属性；游戏币用于培养员工技术。
5. **效率提升**：更高的人气 → 更高的直播收益 → 更多的游戏币 → 更多护送任务 → 更多俱乐部资金 → 更强的俱乐部。

## 二、可行性分析

### 2.1 市场可行性

#### 2.1.1 目标用户画像

| 维度 | 分析 |
|------|------|
| **核心用户** | 搜打撤游戏玩家（塔科夫/暗区/萤火/三角洲等），国内预估 3000 万+活跃受众 |
| **辐射用户** | 微信小游戏泛用户，18-40 岁男性为主，偏好放置/经营类游戏 |
| **用户特征** | 碎片化时间多，偏好"放着不管就能变强"的游戏模式；对搜打撤枪战场景有认知，喜爱圈内文化 |

#### 2.1.2 微信小游戏用户特征匹配

| 微信小游戏用户特征 | 本产品匹配点 |
|-------------------|-------------|
| 即点即玩、无需下载 | 文字+UI 为主的轻量玩法，秒开无压力 |
| 社交裂变驱动 | 俱乐部概念天然适合好友互动、分享炫耀 |
| 碎片化体验 | 派遣任务 1 分钟、收菜 30 秒、看日志 2 分钟 |
| 偏好放置挂机 | 直播收益 = 核心放置机制，离线也能产出 |
| 低 ARPU 但高 DAU | 免费+轻付费设计，大基数支撑收入 |

#### 2.1.3 竞品分析

| 竞品 | 类型 | 差异化 |
|------|------|--------|
| 《叫我大掌柜》 | 模拟经营 | 题材差异——搜打撤题材独一份，圈内文化壁垒 |
| 《动物餐厅》 | 放置经营 | 玩法深度差异——多了培养和策略派遣维度 |
| 《人生模拟器》 | 文字模拟 | 包装差异——固定世界观代入感更强 |
| 《电竞经理》系列 | 卡牌养成 | 玩法更轻松，放置为主，操作更少 |

**市场空白点**: 目前微信小游戏市场尚无"搜打撤题材 + 模拟经营 + 放置挂机"类型的产品，题材壁垒明显。

### 2.2 技术可行性

| 平台 | 优势 | 劣势 | 适配方案 |
|------|------|------|----------|
| **微信小游戏（竖屏）** | 获客成本低、社交裂变强、即点即玩、支付闭环 | 性能受限、不支持 3D、包体限制 | 完全匹配——本游戏以 UI + 文字为主，无 3D 需求 |

**技术方案评估**：

| 技术维度 | 评估 |
|----------|------|
| **渲染方案** | 2D UI + 卡牌/图标 + 文字，微信小游戏完全胜任 |
| **服务器** | 挂机收益计算、事件日志生成、数据存储，常规云服务器即可 |
| **包体大小** | 以 2D 素材为主，预估包体 < 10MB，远低于微信 20MB 限制 |
| **多端适配** | 竖屏设计天然适配手机，无需额外适配 |
| **开发周期** | 2D 竖屏放置游戏，预估 3-4 个月核心版本 |

### 2.3 竖屏适配优势

| 竖屏特性 | 游戏设计对应 |
|----------|-------------|
| 单手操作方便 | 核心操作：收菜、派遣、养成，均为点击操作 |
| 纵向信息流 | 事件日志纵向滚动阅读，天然舒适 |
| 碎片场景友好 | 地铁、排队、午休——单手即可完成所有操作 |
| UI 布局 | 顶部：资源条 / 中部：主界面 / 底部：导航栏 |

### 2.4 商业可行性

| 维度 | 预估 |
|------|------|
| **商业模式** | 免费下载 + 内购（加速道具、外观皮肤、赛季通行证） |
| **主要付费点** | 游戏币加速卡、俱乐部资金礼包、限定员工外观 |
| **ARPU 预期** | 轻度付费为主（月卡 + 通行证），大 R 通过限时活动拉动 |
| **回本周期** | 预估 3-5 个月 |
| **微信生态优势** | 微信支付无缝接入，社交分享自然获客，广告变现可补充 |

### 2.5 风险与对策

| 风险 | 等级 | 对策 |
|------|------|------|
| **内容合规风险** | 🟢 低 | V2 已彻底去灰产化——俱乐部 = 合法服务组织；护送 = 游戏内组队；直播 = 内容创作。详细合规映射见 `00_术语合规映射表.md` |
| **题材接受度** | 🟡 中 | 搜打撤玩家群体大但圈外用户可能不理解背景。首期目标明确锁定圈内用户，后期可拓展泛放置用户 |
| **放置品类留存** | 🟡 中 | 事件日志系统提供每日新鲜感；养成深度提供长期目标；活动系统保持节奏 |
| **微信平台审核** | 🟢 低 | 无社交博弈、无棋牌、无版权风险，纯模拟经营游戏，符合微信小游戏规范 |

## 三、游戏系统架构 V2

### 3.1 核心系统全景

```
《搜打撤俱乐部》系统架构 V2

├── 直播收益系统 —— 放置挂机产出游戏币，受人气影响
├── 护送任务系统 —— 消耗游戏币派遣任务，产出俱乐部资金
├── 员工养成系统 —— 技术培养（游戏币）+ 非技术属性（俱乐部资金）
├── 设备与人气系统 —— 俱乐部资金升级设备 → 提升人气 → 增强全局效率
├── 事件日志系统 —— 每日随机事件，文字叙事，损益反馈
├── 货币管理系统 —— 俱乐部资金 + 游戏币，全俱乐部共用
├── 社交系统 —— 好友俱乐部、排行榜、员工切磋
└── 商业化系统 —— 月卡、通行证、加速道具、外观
```

### 3.2 日循环节奏

| 时段 | 主要行为 | 参考时长 |
|------|----------|----------|
| 上线 | 收取直播收益游戏币、查看事件日志 | ~2 min |
| 随时 | 派遣护送任务（手动选择难度和员工） | ~1 min |
| 随时 | 消耗资源培养员工、升级设备 | ~2 min |
| 下线/挂机 | 直播收益自动持续产出 | - |

### 3.3 用户分层

| 用户类型 | 日时长 | 核心行为 |
|----------|--------|----------|
| 轻度用户 | 3-5 min | 收游戏币、派遣任务、看一眼事件日志 |
| 中度用户 | 10-15 min | 研究最优派遣策略、规划培养路线、管理人气 |
| 重度用户 | 30 min+ | 深度养成、冲排行榜、研究版本内容 |

### 3.4 货币体系

| 货币 | 获取途径 | 消耗途径 | 特性 |
|------|----------|----------|------|
| **游戏币** | 直播收益（放置产出）、事件日志奖励 | 派遣护送任务、培养员工技术 | 挂机持续产出，上限可扩容 |
| **俱乐部资金** | 护送任务奖励、事件日志奖励 | 升级设备、提升人气、培养非技术属性、招募员工 | 珍贵资源，需策略规划 |

### 3.5 员工属性模型

| 属性类别 | 属性名 | 说明 | 培养消耗 |
|----------|--------|------|----------|
| **技术（核心）** | 枪法、意识、身法 | 决定护送任务成功率和收益 | 游戏币，递增成本 |
| **人气（辅助）** | 魅力、口才、颜值 | 影响直播收益效率 | 俱乐部资金 |
| **基础** | 体力、耐久 | 影响持续作战能力 | 俱乐部资金 |
