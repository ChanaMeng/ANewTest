# 护送任务与直播收益系统

> V2重构：移除旧版四阶段战斗模拟（搜刮/战斗/撤离/结算），替换为护送任务派遣 + 直播收益放置 + 随机事件日志。

---

## 一、护送任务系统

### 1.1 系统概述

护送任务是**主动派遣玩法**的核心。玩家消耗"游戏币"派出打手执行护送任务，成功可获得"俱乐部资金"用于基地建设与进阶发展。

**核心循环**：`直播赚游戏币 → 护送花游戏币 → 成功赚俱乐部资金 → 升级基地 → 解锁更高级玩法`

### 1.2 三级难度设计

| 难度 | 消耗游戏币 | 俱乐部资金奖励 | 技术要求 | 额外门槛 | 翻车概率参考 |
|------|-----------|---------------|----------|----------|-------------|
| **初级护送** 🟢 | 200 ~ 500 | 400 ~ 1,000 | 技术 ≥ 30 | 无 | 5% ~ 15% |
| **中级护送** 🟡 | 800 ~ 1,500 | 1,500 ~ 3,000 | 技术 ≥ 60 | 需至少1名★★★以上打手 | 15% ~ 30% |
| **高级护送** 🔴 | 2,000 ~ 5,000 | 4,000 ~ 10,000 | 技术 ≥ 100 | 需特定装备/技能 | 25% ~ 50% |

**翻车后果**：游戏币不退还，打手受伤（休息时间 +N 分钟），无俱乐部资金奖励。

### 1.3 任务池与刷新规则

```
每日任务总数 = 基础3个 + 基地等级加成（每级+1，上限10个）

初级护送：固定2个（保底体验）
中级护送：基础1个 + 声誉解锁（声誉≥100时+1，声誉≥500时+1）
高级护送：声誉≥2000解锁1个，声誉≥5000解锁2个
```

| 刷新方式 | 说明 |
|----------|------|
| **自动刷新** | 每日0点刷新所有未接取任务 |
| **手动刷新** | 消耗"情报碎片×5"或观看广告，刷新单个槽位 |
| **紧急护送** | 随机触发的限时任务，奖励+50%，持续时间30分钟 |

### 1.4 派遣规则

每次护送可派遣 **1~3 名打手**：

| 人数 | 适用场景 | 说明 |
|------|----------|------|
| 1人 | 初级护送、低风险 | 单人实力直接决定成败 |
| 2人 | 中级护送 | 协作加成，技术互补 |
| 3人 | 高级护送 | 满编出击，最大化成功率 |

**协作加成**（仅2~3人派遣时生效）：
- 同一定位重复：效率降低 10%（抢活干）
- 枪男 + 老六 + 车头："搜打撤铁三角"，成功率 +15%
- 枪男 + 气氛组 + 车头："明星车队"，额外游戏币收益 +20%

### 1.5 成功判定机制（简化战斗）

成功不再是四阶段模拟，改为**单次判定**：

```
护送成功率 = 基础成功率 + 技术修正 + 装备修正 + 技能修正 + 随机浮动

基础成功率：
  初级护送：80%
  中级护送：65%
  高级护送：50%

技术修正 = (派遣打手平均技术 / 该难度要求技术) × 20%
  （上限 +25%，下限 -10%）

装备修正：
  每位打手已装备栏位数量 × 2%
  （无装备 = 0%，全装备 = +12%/人）

技能修正：
  - 「安全驾驶」：撤离类技能 +8%
  - 「战术大师」：指挥类技能 +5% 团队
  - 「烟遁」：翻车时 30% 概率免受伤

随机浮动：±10%（薛定谔的护送）
```

**判定示例**：

```
高级护送（要求技术≥100，基础成功率50%）
派遣：锁子哥(技术120) + 老司机(技术110) + 吴姐(技术95)

平均技术 = (120+110+95)/3 = 108
技术修正 = (108/100) × 20% = +21.6%
装备修正 = (3人满装) = 3 × 12% = +36%
技能修正 = 老司机「安全驾驶」+8%
随机浮动 = -3%

总成功率 = 50% + 21.6% + 36% + 8% - 3% = 112.6% → 封顶95%
→ 结果：判定通过！护送成功！+8000俱乐部资金
```

> **设计约束**：成功率硬上限 95%、硬下限 5%，永远保留希望与翻车风险。

### 1.6 翻车惩罚细化

| 翻车类型 | 触发条件 | 惩罚内容 |
|----------|----------|----------|
| 轻度翻车 | 成功率≥70%但判定失败 | 游戏币损失照常，打手心情-10 |
| 中度翻车 | 成功率40%~70%且判定失败 | 游戏币损失照常，随机1人受伤（休息30分钟） |
| 重度翻车 | 成功率<40%且判定失败 | 游戏币损失照常，全员受伤（休息60分钟），声誉-20 |
| 惨案 | 成功率<15%且判定失败 | 以上全部 + 装备耐久-1 + 声誉-50 + 触发负面随机事件 |

## 二、直播收益系统（放置/挂机玩法）

### 2.1 系统概述

直播收益是游戏的**放置（Idle）核心**。打手在「直播室」设施中进行直播，持续产出"游戏币"。这是**被动收入来源**，也是护送任务投入的资源来源。

融合了旧版的"陪玩/直播/代打"概念，统一为"打手开直播"这一放置行为。

### 2.2 基础收益公式

```
直播收益（游戏币/分钟）= 基础产出 × 技术系数 × 人气系数 × 装备系数 × 技能系数

基础产出：
  Lv1 直播室：3 游戏币/分钟
  每升1级 +0.6 游戏币/分钟

技术系数 = 1.0 + (打手技术值 / 100) × 0.5
  （技术100的打手 = 1.5x）

人气系数 = 1.0 + (俱乐部人气 / 1000) × 0.3
  （人气值来源：声誉等级、装饰品、活动奖励）

装备系数 = 1.0 + 装备品质系数
  白装+0.05 / 绿装+0.10 / 蓝装+0.15 / 紫装+0.20 / 金装+0.30 / 红装+0.50
  （取打手最高品质装备）

技能系数（特殊直播技能）：
  「姐姐带你」：直播收益 ×1.25
  「甜言蜜语」：负面事件50%概率转为正面
  「喋喋不休」：直播收益 ×1.10
```

**收益示例**：

```
锁子哥（技术120，金装武器）在 Lv5 直播室直播：
基础产出 = 3 + 5×0.6 = 6/min
技术系数 = 1.0 + 1.2×0.5 = 1.6
人气系数 = 假设俱乐部人气800 → 1.0 + 0.8×0.3 = 1.24
装备系数 = 1.0 + 0.30 = 1.30
技能系数 = 无特殊直播技能 = 1.0

每分钟收益 = 6 × 1.6 × 1.24 × 1.30 × 1.0 = 15.5 游戏币/min
每小时 ≈ 930 游戏币/h
```

### 2.3 直播室设施升级

| 等级 | 名称 | 升级消耗（俱乐部资金） | 基础产出/min | 额外效果 |
|------|------|----------------------|-------------|----------|
| 1 | 地下室直播间 | - | 3 | 可安排1人直播 |
| 2 | 简易直播间 | 500 | 3.6 | 可安排1人 |
| 3 | 标准直播间 | 1,500 | 4.2 | 可安排2人 |
| 5 | 高级直播间 | 5,000 | 5.4 | 可安排2人 |
| 7 | 专业直播间 | 15,000 | 6.6 | 可安排3人 |
| 10 | 顶级直播基地 | 50,000 | 8.4 | 可安排3人 + 人气×1.5 |

**升级限制**：直播室等级 ≤ 基地等级。

### 2.4 多打手同播机制

当设施支持多人同播时，收益叠加计算：

```
总直播收益 = Σ(每人每分钟收益)

但存在"观众分流"惩罚：
  2人同播：每人收益 × 92%（总产出 = 1.84x单人）
  3人同播：每人收益 × 85%（总产出 = 2.55x单人）
```

### 2.5 离线收益机制

```
离线收益 = 当前直播收益 × 离线时间 × 离线衰减系数

离线衰减系数：
  VIP 0（免费）：60%
  VIP 1：80%
  VIP 2：100%（无衰减）

离线收益上限：离线时长 ≤ 12小时（超过不计）
```

**离线收益结算**（回登录时触发）：
- 弹窗展示离线期间总收益
- 可观看广告 ×2 领取翻倍离线收益
- 离线期间触发的随机事件在回到游戏时批量展示

## 三、随机事件系统

### 3.1 系统概述

随机事件贯穿护送任务和直播收益两套系统，以**人生模拟器风格的日志**形式展示。事件每天随机触发，有正有负，直接影响游戏币或俱乐部资金。

### 3.2 护送任务事件

| 事件类型 | 示例 | 效果 |
|----------|------|------|
| **超常发挥** 🔥 | "锁子哥今天手感火热，一枪一个小朋友！护送大成功！" | 俱乐部资金奖励 ×1.5 |
| **意外之喜** 🎁 | "途中发现隐藏物资箱，老司机将其收入囊中。" | 额外获得随机道具/装备 |
| **临危不乱** 🛡️ | "遭遇伏击时三角洲队长冷静指挥，化险为夷。" | 降低失败概率，本次增加免伤 |
| **回头客** 💌 | "委托人很满意，介绍了新客户。" | 立即刷新一个高级护送任务 |
| **突发变故** ⚠️ | "护送途中遭遇强敌，猛攻哥受伤，任务失败！" | 游戏币损失翻倍，打手受伤 |
| **半路截胡** 👤 | "可疑对手试图半路截胡，装备差点被打掉！" | 装备耐久-1，成功率临时-20% |
| **信息泄露** 📢 | "委托信息被泄露，同行竞争加剧。" | 后续3次护送成功率-10% |
| **天气恶劣** 🌧️ | "突发暴雨，可视度极低，撤离变得困难。" | 基础成功率-15%（仅本次） |

### 3.3 直播收益事件

| 事件类型 | 示例 | 效果 |
|----------|------|------|
| **人气爆发** 🎉 | "主播[锁子哥]今天状态超常发挥，直播收入翻倍！" | 该打手直播收益 ×2，持续1小时 |

| **节奏风波** 💬 | "弹幕带节奏，[话痨]跟观众吵起来了，直播间被警告。" | 该打手直播收益-50%，持续30分钟 |
| **技术故障** 🔧 | "直播设备坏了，[打工仔]手足无措。" | 该打手停止产出10分钟 |
| **连麦事故** 🎤 | "[萌新]不小心说错话，弹幕炸了！" | 人气临时-20%，持续20分钟 |

  每30分钟：40% 概率触发1个随机事件
  每天事件上限：护送6个 + 直播12个

## 六、新旧系统对照

| 维度 | 旧版（V1） | 新版（V2） |
|------|-----------|-----------|
| 核心玩法 | 委托订单 + 4阶段战斗模拟 | 护送派遣 + 直播放置 + 随机事件 |
| 战斗表现 | 搜刮→战斗→撤离→结算（4阶段动画） | 单次成功/失败判定（即时） |
| 收益模式 | 委托直接给信用点（主动刷） | 直播挂机产游戏币 → 护送赚俱乐部资金 |
| 玩家节奏 | 需要频繁操作、观看战斗动画 | 直播挂机被动收菜 + 护送定时派遣 |
| 适用场景 | 重度玩家 | 轻度/中度玩家，适合微信小游戏 |
| 随机性 | 战斗中的概率事件（复杂） | 日志式随机事件（轻量透明） |
| 付费设计 | 加速卡跳过战斗 | VIP离线收益、情报刷新、幸运符 |
