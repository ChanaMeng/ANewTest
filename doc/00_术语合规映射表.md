# 术语合规映射表 V2

> V2 重构将游戏定位从"灰色地带打手基地"调整为"合法俱乐部服务平台"，大幅减少合规风险。以下为 V2 版本的术语规范化对照。

---

## 一、核心术语定义（V2 新增）

| 术语 | 英文 | 定义 | 游戏功能 |
|------|------|------|----------|
| **俱乐部资金** | Club Funds | 俱乐部运营资金，等价于 RMB 的概念 | 用于升级设备、提升人气、培养员工非技术属性；通过护送任务获取 |
| **游戏币** | Game Currency | 搜打撤游戏中的虚拟货币（类似哈夫币/科恩币） | 用于派遣护送任务、培养员工技术属性；通过直播收益获取 |
| **护送任务** | Escort Mission | 俱乐部为委托人提供的战术护送服务 | 核心收入玩法，分初级/中级/高级三档，消耗游戏币，产出俱乐部资金 |
| **直播收益** | Streaming Revenue | 员工直播搜打撤游戏产生的持续收益 | 放置/挂机核心玩法，离线持续产出游戏币 |
| **人气** | Popularity | 俱乐部及员工在圈子内的知名度 | 影响直播收益效率和护送任务成功率；通过设备升级提升 |
| **技术** | Skill | 员工的游戏技术水平 | 核心养成维度，通过消耗游戏币培养，层级递增成本 |
| **事件日志** | Event Log | 每日随机事件的文字记录 | 类似《人生模拟器》的叙事方式，展示派遣和直播中的实际损益 |

## 二、V2 术语对照表（设计术语标准化）

> 以下对照将 V2 设计中的游戏化概念与真实世界概念一一对应，确保不触碰合规红线。

| 真实概念 | V2 游戏术语 | 说明 |
|----------|------------|------|
| 人民币 | **俱乐部资金** | 游戏内等价 RMB 的虚拟货币，不直接与现实货币挂钩 |
| 游戏内货币（哈夫币/科恩币） | **游戏币** | 搜打撤游戏内的虚拟货币，用于派遣和养成 |
| 代打/代练 | **护送任务** | 重构为"护送委托人完成搜打撤任务"的合法服务，三档分级 |
| 陪玩 | **直播收益**（挂机产出） | 陪玩概念吸收进直播收益系统，作为放置产出的一部分 |
| 主播代打 | **直播收益**（挂机产出） | 主播相关活动统一归入直播收益系统 |
| 打手/雇佣兵 | **员工** | 去军事化，改用企业化称谓 |
| 老板/客户 | **委托人** | 保留，中性化表达 |
| 工会/战队 | **俱乐部** | 保留，类似"公会"的企业化表达 |
| 基地/工作室 | **俱乐部** | 统一使用"俱乐部"，去"地下"暗示 |

## 三、合规红线规避清单

> V2 设计中已主动规避以下敏感概念，但仍需保持警惕，确保不通过任何 UI 文案/事件文本/图标产生映射。

| 严格禁止出现的概念 | 风险等级 | 替代方案 |
|-------------------|---------|----------|
| "代打""代练"等词汇 | 🔴 高 | 已统一为【护送任务】【直播收益】 |
| "护航"（作为灰产术语） | 🔴 高 | 已统一为【护送任务】，仅保留"护送"作为任务描述动词 |
| "老板/金主"（有雇佣灰产暗示） | 🟡 中 | 统一使用【委托人】 |
| "灰色/地下/隐秘"等修饰 | 🟡 中 | 不出现此类描述，世界观定位为正当俱乐部 |
| "外挂/科技选手" | 🟡 中 | 替换为【可疑对手/异常遭遇】等虚构事件名 |
| "封号/毁号" | 🟡 中 | 替换为【封禁风波】等自创事件名 |
| "大红/大金"（真实游戏术语） | 🟢 低 | 替换为【至宝/极品物资】等架空名称 |
| 真实游戏名称/平台名称 | 🔴 高 | 不出现任何真实游戏名（三角洲行动/暗区突围等） |

## 四、保留术语清单

> 以下搜打撤圈子通用文化术语可安全使用，无灰产映射风险：

| 术语 | 含义 | 保留理由 |
|------|------|----------|
| **搜打撤** | Search-Fight-Extract 品类描述 | 品类通用词，无灰产映射 |
| **俱乐部** | Club，玩家组织形式 | 中性词，类似"公会" |
| **翻车** | 任务失败 | 通用网络用语 |
| **车头** | 车队领导者 | 游戏通用术语 |
| **老六** | 偷袭者 | 游戏通用术语 |
| **猛攻** | 激进打法 | 游戏玩法描述 |
| **撤离** | Extract | 品类核心动作 |

## 五、世界观合规声明

```
V1 版本：
  "玩家扮演灰色地带俱乐部老板，经营护航/代练/陪玩业务"

V2 版本：
  "玩家扮演搜打撤俱乐部的经营者，通过员工直播收益和护送任务
   获取资源，打造圈内最受欢迎的俱乐部服务平台"
```

**V2 定位原则**：
1. 俱乐部 = 合法服务平台，提供战术协助和娱乐内容
2. 护送任务 = 游戏内合法的组队服务，非代打
3. 直播收益 = 内容创作收益，非灰色陪玩
4. 所有文案不出现任何暗示现实灰产的表述
