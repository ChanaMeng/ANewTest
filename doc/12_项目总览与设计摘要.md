## 一、执行摘要 (Executive Summary)

《搜打撤俱乐部》是一款面向微信小游戏平台的**竖屏模拟经营放置手游**。游戏以"搜打撤"（Extraction Shooter）游戏生态为背景，玩家扮演一家服务俱乐部的经营者，通过派遣打手执行护送任务、运营直播业务来获取收益。

**核心差异化**：目前市面上尚无直接竞品。"搜打撤"品类国内受众超 5000 万（《三角洲行动》国服日活突破 4100 万），且《电竞经理》系列已验证"圈内梗 + 卡牌养成"模式的商业可行性。本作将搜打撤生态中的"俱乐部经营"这一独特视角游戏化，填补了模拟经营品类的空白。

**V2 版本核心理念**：双货币经济循环驱动核心玩法 —— 俱乐部资金（RMB 等价）用于装备购买、人气投资、员工非技术属性提升；游戏币（类哈夫币/科恩币）用于员工技术培养，是游戏的核心成长资源。玩家通过派遣员工执行护送任务消费游戏币、获取俱乐部资金，同时直播系统作为放置收入持续产出游戏币，形成"投入→产出→再投入"的闭合经营循环。

**商业模式**：免费下载 + 激励广告 + IAP，微信小游戏社交裂变获客，预期 ARPU 中等偏上，3-6 个月回本。

## 二、一页纸设计总览 (One-Page Design Overview)

| 系统 | 核心设计 | 关键数值 |
|------|----------|----------|
| **双货币经济** | 俱乐部资金（RMB 等价）+ 游戏币（类哈夫币/科恩币） | 俱乐部资金：装备/人气/非技术属性；游戏币：技术培养/主进度 |
| **护送任务** | 消费游戏币派遣员工，产出俱乐部资金 | 3 个难度档位：初级/中级/高级 |
| **直播收益** | 放置挂机产出游戏币（合并原陪玩/直播/代打） | 挂机收益 × 人气系数 |
| **人气系统** | 俱乐部人气影响收益倍率 | 装备升级提升人气上限 |
| **技术培养** | 员工核心养成属性，游戏币驱动 | 分层指数成本曲线 |
| **随机事件** | 人生模拟器风格日志展示 | 双向影响两种货币 |
| **员工卡牌** | 5 种定位 + 5 档稀有度 + 6 维属性 | 王牌（★★★★★）至新人（★★） |
| **基地设施** | 8 种设施，各自影响经营效率 | 训练房/装备库/情报中心/直播室等 |
| **赛季系统** | 12 周赛季 + 通行证 + 对抗赛 | 赛季限定角色/皮肤 |
| **活动系统** | 猛攻节（月度）+ 限定事件链 | 收益翻倍，限定奖励 |
| **商业化** | 抽卡 + 通行证 + 月卡 + 激励广告 + IAP | 月卡 ¥30-98，通行证 ¥68-128 |
| **社交** | 好友 + 借调 + 排行榜 + 微信群分享 | 微信小游戏社交能力全面接入 |

## 五、核心创新点 (Core Innovation Points)

### 5.1 双货币闭环经济

区别于传统放置游戏"单货币线性产出"模式，本作采用 **俱乐部资金 + 游戏币** 双货币体系：
- 两种货币通过不同系统获取、投入不同成长维度
- 形成"消费游戏币派遣 → 收获俱乐部资金 → 再投资扩大效率"的正向循环
- 玩家需在"当前收益"与"长期投资"之间做决策，增加策略深度

### 5.2 搜打撤生态独特视角

- 市面上搜打撤游戏都聚焦于"FPS 射击"本身，本作为**首款**从俱乐部经营者角度切入的模拟游戏
- 还原圈内真实经济模型（俱乐部资金 ↔ 游戏币兑换），让搜打撤玩家获得"经营方"视角的代入感
- 丰富的圈内梗和事件文案，打造强情感共鸣

### 5.3 人生模拟器风格事件系统

- 采用"日志流"展示随机事件，类似《人生模拟器》的阅读体验
- 事件不只是数值变动，而是迷你叙事片段，增强沉浸感
- 事件链设计（多日连锁选择），提供分支叙事和策略抉择

### 5.4 技术属性分层培养

- 将员工养成聚焦于"技术"属性，避免传统卡牌游戏属性膨胀
- 指数级成本曲线确保长期追求，但低级培养零门槛，保持轻度友好
- 技术等级直接决定可参与的护送任务难度，形成清晰的成长目标

### 5.5 放置与策略的平衡

- 轻度用户：上线收菜 + 一键派遣 + 看事件日志，5 分钟完成
- 中度用户：阵容搭配 + 技术培养路线选择 + 任务难度评估
- 重度用户：赛季冲榜 + 对抗赛战术研究 + 社交竞争

| 1 | **货币体系** | 信用点（软货币）+ 暗金（硬货币）+ 声誉徽章 + 友情点（四货币） | 俱乐部资金 + 游戏币（双货币） | 四货币过度复杂，双货币还原真实俱乐部经济模型（RMB ↔ 游戏币兑换），降低认知负担 |
| 2 | **游戏名称** | "打手基地" | "搜打撤俱乐部" | "俱乐部"更契合搜打撤圈真实生态，且更中性化，降低合规风险 |
| 3 | **委托类型** | 8 种委托类型（日常护卫/定向寻宝/主播任务/搭档委托/紧急/高级/赛季/情报） | 护送任务（初级/中级/高级 3 档） | 类型过多造成决策疲劳。合并为 3 档难度，清晰易懂，保留足够策略深度 |
| 4 | **直播/陪玩/代打** | 各自独立的委托/收益系统 | 合并为"直播收益"放置系统 | 三者经济模型相似（技能变现），合并简化受众认知，专注做深放置体验 |
| 5 | **员工养成维度** | 6 维属性 + 等级 + 技能 + 装备 + 突破 + 好感度（6 条养成线） | 技术（核心属性）+ 装备 + 技能（精简） | V1 养成线过多导致付费压力大和认知负担。V2 聚焦"技术"，指数成本曲线做深 |
| 6 | **人气系统** | 无独立人气系统，仅声誉徽章 | 人气作为独立系统，直接驱动收入乘数 | 人气是俱乐部经营的核心指标，独立系统带来更直观的经营反馈感 |
| 7 | **随机事件展示** | 弹窗式事件通知 | 人生模拟器风格日志流 | 日志流阅读体验更强，符合"经营日志"的叙事代入感，支持事件链展示 |
| 8 | **收入来源** | 多系统分散产出（委托/直播/陪玩/代打） | 双系统集中产出（护送任务 → 俱乐部资金，直播 → 游戏币） | 简化玩家认知路径：派遣耗币赚钱，挂机产币养人 |
| 9 | **技术成本曲线** | 阶梯式（配置固定消耗） | 分层指数增长 | 指数成本确保长期养成追求，避免快速毕业。分层设计保证新手体验平滑 |
| 10 | **商业化驱动** | 抽卡 + 暗金充值为主 | 激励广告 + 抽卡 + IAP 并行 | 微信小游戏天然适合激励广告模式，轻付费 + 重广告变现符合平台生态 |
