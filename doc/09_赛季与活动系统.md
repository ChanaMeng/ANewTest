# 赛季与活动系统

## 一、赛季系统

### 1.1 赛季基础结构

| 项目 | 设定 |
|------|------|
| 赛季时长 | 12周（约3个月） |
| 赛季主题 | 每赛季更换，与搜打撤热门事件联动 |
| 赛季版本号 | S1 / S2 / S3... |
| 赛季内容 | 新员工、新委托地图、新设施外观、新活动类型 |
| 赛季结算 | 第12周周日23:59:59 |
| 休赛期 | 结算后48小时，不可参与排名活动 |

### 1.2 赛季节奏设计（12周）

| 周次 | 阶段 | 内容 |
|------|------|------|
| 第1周 | 赛季开启 | 新赛季内容上线、通行证开售、排名重置 |
| 第2-3周 | 成长期 | 猛攻节（月度活动第1轮）、新员工限时招募 |
| 第4周 | 冲刺期 | 双倍直播周、护送挑战 |
| 第5周 | 赛季中期 | 猛攻节（月度第2轮）、人气冲刺活动 |
| 第6-7周 | 深化期 | 限定事件链、俱乐部对抗赛锦标赛 |
| 第8周 | 赛季后期 | 猛攻节（月度第3轮）、回流活动 |
| 第9-10周 | 冲刺期 | 双倍奖励周、全服BOSS活动 |
| 第11周 | 赛季收官 | 赛季结算预告、排名冲刺活动 |
| 第12周 | 结算周 | 赛季结算、休赛期、下赛季预告 |

### 1.3 赛季主题示例

| 赛季 | 主题名称 | 主题风格 | 限定内容 |
|------|----------|----------|----------|
| S1 | 「新手启航」 | 军事基地风 | 基础内容上线 |

| S5 | 「都市暗影」 | 赛博朋克风 | 夜城地图 + 未来科技员工 |
| S6 | 「战火重燃」 | 战争废墟风 | 阵营对抗 + 史诗委托链 |

### 1.4 赛季重置机制

| 系统 | 是否重置 | 重置规则 |
|------|----------|----------|
| 对抗赛段位 | 部分重置 | 传说→钻石 / 大师→铂金 / 钻石→黄金 / 铂金→白银 / 其他→青铜 |
| 排行榜 | 完全重置 | 所有排名归零，新赛季重新计算 |
| 赛季积分 | 完全重置 | 赛季积分清零 |
| 通行证进度 | 完全重置 | 新赛季新通行证，从Lv1开始 |
| 员工/养成进度 | 不重置 | 永久保留 |
| 设施等级 | 不重置 | 永久保留 |
| 俱乐部资金/游戏币 | 不重置 | 永久保留（部分货币赛季币清零） |
| 好感度 | 不重置 | 永久保留 |
| 仓库/装备 | 不重置 | 永久保留 |

### 1.5 赛季特殊货币

| 货币 | 获取方式 | 用途 | 赛季结束后 |
|------|----------|------|-----------|

## 三、猛攻节（月度核心活动）

### 3.1 猛攻节基础设定

猛攻节是每月的核心大型活动，模拟搜打撤游戏中"猛攻"（高强度战斗）的狂欢氛围。

| 项目 | 设定 |
|------|------|
| 开启时间 | 每月1日 10:00 - 每月7日 23:59（7天） |
| 参与条件 | 作战指挥室Lv5以上 |
| 主题轮换 | 每月不同主题（BOSS挑战/精英委托/资源冲刺） |
| 核心玩法 | 完成猛攻任务 → 积累猛攻积分 → 解锁奖励 |

### 6.3 活动密度控制原则

```
活动密度控制原则：
1. 每周至少1个在运行的活动
2. 重要节日（春节/国庆）预留至少3天集中体验
3. 避免猛攻节与节日活动时间重叠
4. 周末固定保留双倍直播活动
5. 赛季末周不启动新活动（为结算留空间）
6. 每月保持7-10天活动空窗期（让玩家消化日常内容）
```
