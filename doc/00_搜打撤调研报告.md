# 搜打撤俱乐部调研报告

> 调研时间：2026年5月  
> 调研目的：为《搜打撤俱乐部》模拟经营游戏提供品类理解与设计参考

---

## 一、什么是"搜打撤"

### 1.1 品类定义

"搜打撤"是中国玩家对**撤离类射击游戏（Extraction Shooter）**的俗称，属于射击游戏的新兴子品类，被视为继"大逃杀"之后的又一现象级风口。

名称源自核心玩法的三个阶段：

| 阶段 | 含义 | 玩家行为 |
|------|------|----------|
| **搜**（Search） | 搜刮物资 | 在地图内搜索武器、装备、高价值物品 |
| **打**（Fight） | 战斗对抗 | 与AI敌人或其他玩家交战，争夺资源 |
| **撤**（Extract） | 安全撤离 | 携带战利品到达撤离点，保住收益 |

### 1.2 与大逃杀的差异化

| 维度 | 搜打撤 | 大逃杀 |
|------|--------|--------|
| **胜利条件** | 活着撤出即为赢（分层胜利） | 最终吃鸡唯一胜利 |
| **经济系统** | 持续性局外资产积累（仓鼠式理财） | 单局用完即弃 |
| **风险博弈** | 带入装备可能全丢，撤出则永久保留 | 无局外资产风险 |
| **玩家类型** | 全装大佬/跑刀仔 并存 | 统一目标 |

### 1.3 核心吸引力

- **"出金"快感**：搜到高价值物品并成功撤出的多巴胺冲击
- **风险博弈**：每次出击都是"投入 vs 产出"的心理博弈
- **长线积累**：仓库资产的RPG式成长驱动力
- **分层胜利**：不同装备投入对应不同胜利标准，新手也能快乐

## 二、代表作品

### 2.1 品类鼻祖：《逃离塔科夫》（2016）

- 将FPS与深度RPG元素粗暴融合
- 极高死亡惩罚：角色死亡装备全丢
- 晦涩系统+极短TTK（击杀时间）
- 定位：硬核玩家的"白月光与受苦梦魇"
- 影响：定义了品类基本范式，但因门槛过高长期局限于小众圈层

### 2.2 现象级爆款：《三角洲行动》

- 开发商：腾讯天美J3工作室
- 国服日活突破4100万
- 在保留硬核撤离机制的同时大幅降低门槛
- 干员技能系统增加差异化
- **游戏内核心货币：哈夫币** — 用于购买装备、武器、消耗品

### 2.3 门槛突破者：《暗区突围》

- 继承"资源运营+战术撤离"核心机制
- 通过保姆级引导、紧凑交火节奏、优化操作逻辑打破品类壁垒
- **游戏内核心货币：科恩币** — 游戏内经济系统的基石，用于交易行、商人兑换等

### 2.4 其他竞品

| 游戏 | 特点 |
|------|------|

## 三、搜打撤俱乐部生态调研

### 3.1 什么是"俱乐部"

搜打撤俱乐部是依托于搜打撤游戏生态的**玩家服务组织**，类似传统网游的"公会/工作室"，为付费玩家提供各类游戏内服务。俱乐部通常以微信/QQ群、Discord或独立网站为运营载体。

### 3.2 俱乐部核心业务

| 业务 | 说明 | 收费模式 |
|------|------|----------|
| **护航** | 精锐玩家带队保护委托人完成搜打撤全流程，确保安全撤离并收获高价值物资 | 按小时/按局收费，约80-300元/小时 |
| **陪玩** | 与委托人组队游戏，提供陪伴式游戏体验 | 按小时收费，30-100元/小时 |
| **代打/代练** | 替代委托人在其账号上完成高难度任务或刷取游戏币 | 按任务/按量收费 |
| **直播** | 俱乐部成员通过直播平台展示高光操作，获取打赏与流量收益 | 平台分成+打赏 |

### 3.3 俱乐部人员架构

```
俱乐部老板/经营者
  ├── 打手团队（护航/代打核心战力）
  │   ├── 王牌打手：技术顶尖，价格最高
  │   ├── 主力打手：稳定输出，中坚力量
  │   └── 新人打手：培养中，价格亲民
  ├── 陪玩团队
  ├── 直播/内容团队
  └── 后勤/客服
```

### 3.4 俱乐部收入模型

- **护航/代打占比最高**，是俱乐部主要营收来源
- **直播收入**具有长尾效应和品牌曝光价值
- **会员制**：部分俱乐部采用月费/季度费会员模式
- **灰色地带**：部分俱乐部涉及外挂等不正当竞争，游戏厂商持续打击

### 3.5 玩家文化现象

| 圈内术语 | 含义 |
|----------|------|
| **跑刀仔** | 只带一把刀入局，零成本博高收益 |
| **全装大佬** | 全身高级装备，实力碾压 |
| **大金/至宝** | 极高价值的稀有物品 |
| **翻车** | 任务失败，装备全丢 |
| **老六** | 善于埋伏、偷袭的玩家 |
| **车队/车头** | 组队核心，负责指挥与决策 |
| **猛攻** | 放弃保守策略，主动出击 |

## 四、对游戏设计的启发

### 4.1 核心代入感

游戏应还原"俱乐部经营者"的真实体验感：
- 资金（俱乐部资金）投入 vs 游戏币产出 的经济博弈
- 不同级别打手的能力差异与培养价值
- 随机事件带来的经营不确定性

### 4.2 货币系统动机

真实俱乐部生态中存在明确的"RMB ↔ 游戏币"兑换关系：
- 俱乐部用RMB雇佣打手、运营日常
- 打手产出游戏币（护航/代打报酬）
- 差额即为俱乐部利润

这正是本游戏"**俱乐部资金（RMB）+ 游戏币（哈夫币/科恩币）**"双货币设计的现实依据。

### 4.3 玩家情感共鸣点

| 共鸣点 | 游戏化方向 |
|--------|------------|
| 抽到/培养"王牌打手"的成就感 | 卡牌养成 |
| 翻车/烂队友的槽点 | 随机事件笑点 |
| 爆出大金的狂喜 | 护送任务暴击奖励 |
| 同行竞争的江湖感 | 对抗赛/排行榜 |
| 圈内梗的认同感 | 事件文案/角色梗 |
